# Sublime Text 3 配置文件
<!-- @author DHJT 2018-05-23 -->

## CovertToUTF8
Sublime Text默认是只支持UTF8的编码，当打开GBK文件时候，文件内会出现部分的乱码，装上这个插件即可支持GBK编码了。

## Emmet
前身是 Zen Coding。它让编写 HTML 代码变得简单。
用法：输入代码后，按Tab或Ctrl+e

## Bracket​Highlighter
括号、引号、标签高亮

## SublimeTmpl
新建模板
ctrl+alt+h 新建html文件
ctrl+alt+j 新建js文件
ctrl+alt+c 新建css文件
如果想编辑默认模版，到"SublimeTmpl\templates"文件夹修改
